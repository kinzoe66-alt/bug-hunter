# Runtime

This folder contains the runtime components of AppSpec:

- **handlers.py** - Unified capability execution.
- **runner.py** - Core loop running the execution plan.
- **audit.py** - Execution auditing and lifecycle tracking.
- **memory.py** - Runtime memory store.
- **result.py** - Standardized execution result class.
- **phase_resolver.py** - Resolves the current phase from plan.
- **execution_plan.py** - Materializes execution phases from appspec.
- **pipeline.py** - Optional execution orchestration utilities.
- **capability_schema.py** - Schema-based validation helpers.
- **replay.py** - Deterministic replay validation.
